# Скролл интерфейса

Добавлены правки для корректной работы скролла в веб-интерфейсе.

Что было сделано:
- Верхнее меню (вкладки) теперь не выходит из контейнера на маленьких экранах — при переполнении появляется горизонтальная полоса прокрутки.
- Вся основная область страницы (`TradingInterface`) получает вертикальную прокрутку при переполнении.

Технические изменения:
- В `frontend/src/components/TradingInterface.tsx`:
  - добавлен атрибут `data-testid="top-tabs-wrapper"` на обёртку всплывающих вкладок и класс `overflow-x-auto` для горизонтальной прокрутки;
  - добавлен `whitespace-nowrap` к контейнеру вкладок, чтобы кнопки не переносились на новую строку;
  - добавлен `data-testid="page-container"` и класс `overflow-y-auto` для основной обёртки приложения, чтобы включить вертикальный скролл.

Подсказки для тестирования локально:

1. Запустите фронтенд dev-сервер:

```pwsh
cd frontend
npm run dev
```

2. Откройте в браузере `http://localhost:5173` и уменьшите ширину окна — верхнее меню должно стать горизонтально прокручиваемым.

3. Если необходимо — добавьте большое количество карточек или элементов на страницу, чтобы проверить вертикальный скролл.
